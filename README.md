# Udaya TK - Professional Portfolio

A modern, responsive portfolio website showcasing professional skills, experience, and projects.

## Features

- **Responsive Design**: Works seamlessly on all devices (desktop, tablet, mobile)
- **Modern UI/UX**: Clean, professional design with smooth animations
- **Interactive Elements**: Hover effects, smooth scrolling, and dynamic content
- **Fast Loading**: Optimized performance with efficient CSS and JavaScript
- **Accessible**: Built with accessibility best practices

## Sections

1. **Hero Section**: Professional introduction with call-to-action buttons
2. **About**: Personal background and key statistics
3. **Skills**: Technical skills organized by categories
4. **Experience**: Professional timeline with detailed descriptions
5. **Projects**: Showcase of featured projects with technologies used
6. **Contact**: Contact information and functional contact form

## Technologies Used

- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **Styling**: CSS Grid, Flexbox, CSS Animations
- **Icons**: Font Awesome
- **Fonts**: Google Fonts (Inter)
- **Features**: Intersection Observer API, CSS Custom Properties

## Setup and Usage

1. Clone or download the repository
2. Open `index.html` in your web browser
3. Customize the content with your personal information:
   - Update personal details in `index.html`
   - Modify colors and styling in `styles.css`
   - Add your own projects and experience

## Customization

### Personal Information
Edit the following sections in `index.html`:
- Hero section title and description
- About section content
- Skills and technologies
- Professional experience
- Project details
- Contact information

### Styling
Modify `styles.css` to customize:
- Color scheme (main colors: #667eea, #764ba2)
- Typography and spacing
- Animation timing and effects
- Layout and component styling

### Functionality
Update `script.js` for:
- Form handling (add backend integration)
- Animation preferences
- Interactive behaviors
